# 会务管理系统 — 接口设计文档

> 版本：v1.0  
> 日期：2026-05-25

---

## 约定

- **Base URL**：`http://localhost:8088/meeting/`（开发环境）
- **请求方式**：GET 用于读取，POST 用于写入
- **返回格式**：统一 JSON `{"code": 200, "msg": "success", "data": {...}}`
- **认证方式**：Session（登录后 `req.getSession().setAttribute("user", user)`）
- **编码**：全部 UTF-8

### 统一响应格式

```json
{
    "code": 200,        // 200成功, 400参数错误, 401未登录, 403无权限, 500服务端错误
    "msg": "操作成功",
    "data": {}          // 具体数据，可为对象、数组或 null
}
```

## 用户系统

### 1. 注册

| 项目 | 内容                                                          |
|------|-------------------------------------------------------------|
| URL | `/user/register`                                            |
| 方法 | POST                                                        |
| 参数 | `username`, `password`,  `phone`, `email`                   |
| 成功返回 | `{"code":200, "msg":"您的账户已成功创建，正在跳转到登录页面...", "data":null}` |
| 失败返回 | `{"code":300, "msg":"注册失败", "data":null}`                 |
| 失败返回 | `{"code":400, "msg":"用户名已存在", "data":null}`                 |
| 失败返回 | `{"code":500, "msg":"注册用的手机或邮箱已存在", "data":null}`                 |


### 2. 登录

| 项目 | 内容 |
|------|------|
| URL | `/user/login` |
| 方法 | POST |
| 参数 | `username`, `password` |
| 成功返回 | `{"code":200, "msg":"登录成功", "data":{"id":1, "username":"admin", "role":"admin"}}` |
| 失败返回 | `{"code":400, "msg":"用户名或密码错误", "data":null}` |

### 3. 退出

| 项目 | 内容 |
|------|------|
| URL | `/user/logout` |
| 方法 | GET |
| 参数 | 无 |
| 返回 | 重定向到 `login.jsp` |

### 5. 修改个人信息
