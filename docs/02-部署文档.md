## 3. 数据库部署

### 3.1 创建数据库

```sql
CREATE DATABASE web_db DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE web_db;
```

### 3.2 执行建表脚本

依次创建：users → conferences → attendees → payments → checkins → accommodations

### 3.3 初始化管理员账号

```sql
INSERT INTO users (username, password, role, status)
VALUES ('admin', MD5(CONCAT('admin123', 'your_salt')), 1, 1);
```

### 3.4 数据库连接配置

修改 `JDBCUtil.java`：

- DB_URL：数据库地址和库名
- DB_USER：数据库用户名
- DB_PASSWORD：数据库密码

## 4. 构建与部署

### 4.1 Maven 构建

```bash
cd web_project
mvn clean package
```

生成的 WAR 包位于 `target/web_project.war`

### 4.2 部署到 Tomcat

将 WAR 包复制到 Tomcat `webapps/` 目录，启动 Tomcat 自动解压：

```bash
cp target/web_project.war /path/to/tomcat/webapps/
/path/to/tomcat/bin/startup.bat    # Windows
/path/to/tomcat/bin/startup.sh     # Linux
```

### 4.3 IDEA 直接运行

1. Run → Edit Configurations
2. 添加 Tomcat Server → Local
3. Deployment 添加 `web_project:war exploded`
4. Application context 设为 `/`
5. 点击运行

### 4.4 访问地址

| 页面 | URL |
|------|-----|
| 用户首页 | http://localhost:8080/index2.jsp |
| 管理员首页 | http://localhost:8080/index1.jsp |
| 登录页 | http://localhost:8080/login.jsp |
| 注册页 | http://localhost:8080/register.jsp |

## 7. 常见问题

| 问题 | 原因 | 解决方式 |
|------|------|----------|
| Access denied for user root | 数据库密码错误 | 检查 JDBCUtil 中密码 |
| ClassNotFoundException: Driver | MySQL 驱动未加载 | 检查 pom.xml，重新构建 |
| 端口 8080 被占用 | Tomcat 未正常关闭 | 结束 java 进程或改端口 |
| 页面 404 | WAR 未正确部署 | 检查 webapps 目录和 URL |
| LocalDateTime 序列化报错 | 未注册模块 | mapper.registerModule(new JavaTimeModule()) |
| JSP 样式丢失 | 浏览器缓存 | Ctrl+F5 强制刷新 |
